## 📖 Sobre o Projeto

O Kiyan foi desenvolvido para oferecer uma experiência moderna, intuitiva e responsiva para apresentação de produtos, informações institucionais e gerenciamento administrativo.

O projeto possui uma interface elegante focada na experiência do usuário, permitindo navegação fluida entre produtos, informações técnicas e conteúdo corporativo.

---

## ✨ Funcionalidades

### 🌐 Landing Page

- Home moderna e responsiva
- Design clean e profissional
- Navegação intuitiva
- Carrossel de produtos

### 📦 Catálogo de Produtos

- Exibição dinâmica de produtos
- Visualização detalhada

- Variações de modelos
- Galeria de imagens

### 🔐 Área Administrativa

- Sistema de login
- Controle de acesso

- Gerenciamento de conteúdo
- Interface exclusiva para administradores

